This folder is a **ready-to-contribute** copy of the PinRAG skill for [block/agent-skills](https://github.com/block/agent-skills).

**To submit to the Goose Skills Marketplace:**

1. Fork [block/agent-skills](https://github.com/block/agent-skills).
2. Copy this `pinrag` directory to the **root** of your fork (next to `code-review/`, `api-setup/`, etc.).
3. Open a pull request using the repository’s “Submit a Skill” template.

Validation runs in CI; `name` in `SKILL.md` frontmatter must match the folder name `pinrag`.
